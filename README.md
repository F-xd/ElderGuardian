# 智慧养老院系统 - 个人作品介绍

## 项目概述

智慧养老院系统（ElderGuardian）是一个集硬件、软件于一体的综合性养老管理解决方案，旨在通过智能化技术提升养老院的管理效率和老人的生活质量。系统实现了老人健康监测、环境监测、设备管理、告警中心等核心功能，为养老院提供全方位的智慧化管理工具。

## 技术栈

### 前端技术
- **框架**：React 18.2.0
- **构建工具**：Vite 7.2.4
- **UI组件库**：Ant Design 6.1.4
- **状态管理**：Redux Toolkit 2.11.2
- **路由**：React Router 7.12.0
- **图表库**：ECharts 6.0.0
- **HTTP客户端**：Axios 1.13.4
- **时间处理**：Day.js 1.11.19
- **动画效果**：React CountUp 6.5.3

### 后端技术
- **框架**：Spring Boot 4.0.4
- **数据库**：MySQL
- **ORM**：JPA
- **WebSocket**：实时通信
- **认证**：JWT
- **消息队列**：AMQP
- **阿里云IoT**：设备接入与管理
- **工具库**：FastJSON、MapStruct

### 硬件技术
- **主控芯片**：STM32F103
- **传感器**：
  - DHT11（温湿度）
  - MAX30102（心率、血氧）
  - MPU6050（加速度、陀螺仪）
  - MQ2（烟雾、有害气体）
  - ADC（模拟信号采集）
- **通信模块**：ESP8266（Wi-Fi）

## 核心功能

### 1. 老人健康监测
- **实时健康数据采集**：通过MAX30102传感器监测老人心率、血氧等健康指标
- **健康数据统计与分析**：使用ECharts生成健康数据趋势图表
- **异常健康状态告警**：当健康指标超出正常范围时触发告警
- **健康档案管理**：记录老人历史健康数据，支持查询与分析

### 2. 环境监测
- **多维度环境数据**：监测房间温度、湿度、烟雾浓度等环境指标
- **环境数据可视化**：通过图表展示环境数据变化趋势
- **环境异常告警**：当环境指标异常时触发告警
- **环境数据历史记录**：支持查询历史环境数据

### 3. 设备管理
- **智能设备监控**：实时监控设备在线状态
- **设备数据采集**：收集设备运行数据
- **设备远程控制**：支持远程控制部分智能设备
- **设备故障告警**：当设备异常时触发告警

### 4. 告警中心
- **多类型告警**：健康告警、环境告警、设备告警等
- **告警级别分类**：根据紧急程度分类告警
- **告警处理流程**：支持告警确认、处理、记录等流程
- **告警历史查询**：支持查询历史告警记录

### 5. 房间管理
- **房间信息管理**：管理房间基本信息、容量等
- **房间状态监控**：实时监控房间占用情况
- **房间环境监测**：监测房间内环境指标
- **房间分配管理**：支持老人房间分配与调整

### 6. 用户管理
- **多角色管理**：管理员、护理员、家属等角色
- **权限控制**：基于角色的权限管理
- **用户信息管理**：管理用户基本信息
- **密码管理**：支持密码修改等功能

## 技术难点与解决方案

### 1. 硬件与软件集成
- **难点**：硬件设备与后端服务的通信协议设计与实现
- **解决方案**：使用MQTT协议实现设备与云端的通信，通过阿里云IoT平台进行设备管理

### 2. 实时数据处理
- **难点**：大量实时数据的采集、传输与处理
- **解决方案**：使用WebSocket实现前端与后端的实时通信，优化数据传输格式，减少网络负载

### 3. 多传感器数据融合
- **难点**：多种传感器数据的采集、校准与融合
- **解决方案**：设计统一的数据采集框架，实现传感器数据的标准化处理

### 4. 系统安全
- **难点**：确保系统数据安全和用户权限管理
- **解决方案**：实现JWT认证机制，基于角色的权限控制，数据加密传输
